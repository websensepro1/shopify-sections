Step 1: Start with the Latest Version
First things first, you want to make sure you’re working with the newest version of the Horizon theme.

Head over to your Shopify admin panel. Click on Online Store > Themes. If you see an update available for Horizon, click the three little dots and hit Update. Once it’s done, click Publish to make it live.

Step 2: Open the Customizer
Once your theme is live, hit that Customize button.

You’ll see your storefront with the default footer at the bottom. It works, sure, but it doesn’t wow. Let’s fix that.

Step 3: Fix the Utility Bar Spacing
Look closely at the very bottom section—the one with the copyright text. You might notice the top and bottom padding is uneven. It’s a tiny detail, but fixing it makes everything look more polished.

Click on the Utility section. Adjust the padding settings so the text sits perfectly centered vertically. Clean. Simple. Professional.

While you’re there, change the Color Scheme to something that contrasts nicely with the rest of the footer. I like Scheme 5 because it creates a subtle separation, but play around with it. You can even create your own custom colors if you’re feeling adventurous.

Step 4: Add Your Social Links and Policies
Still in the utility section, you’ll see options to add Social Media Links. Plug in your profiles. Below that, you can manage your store policies—terms of service, privacy policy, refund policy. Shopify lets you generate these automatically, which is a huge time saver.

Oh, and if you want to remove the “Powered by Shopify” text from the copyright? Just toggle that option off.

Step 5: Build Out Your Main Footer
Now let’s work on the main footer area. By default, it’s a little plain. We’re going to turn it into a three-column layout.

Click on the Footer section and start adding blocks.

Add a Menu Block: This will hold your footer links (like About Us, Contact, FAQ). If you haven’t created a separate footer menu in your Shopify backend yet, you’ll want to do that first. But for now, you can use the main menu to test things out.
Add a Text Block: This is where you can put your address or contact info.
Use the Email Signup Block: Keep this for your newsletter. Add a catchy heading like “Subscribe and get 10% off.”
Step 6: Align Everything (The Secret Sauce)
Here’s where things can get a little messy if you’re not careful. When you add blocks, the headings might not line up perfectly.

Click on the Group that holds your menu and address. Set the alignment to Top and remove any extra gap (set it to 0 pixels). If the address block still looks slightly off, click into that specific block and add a tiny bit of top padding—say, 10 pixels—to nudge it down.

Once that’s done, all three headers should line up like soldiers.

Step 7: Change the Background Color
With everything aligned, click back into the main footer section and change the Color Scheme. I’m a fan of Scheme 3 for this, but pick whatever fits your brand.

This simple color shift creates a visual hierarchy between the main footer and the utility bar at the bottom. It just feels more intentional.

Step 8: Add a Shape Divider (The Fun Part)
Now for the cherry on top: a shape divider. This is that little wavy or jagged line that separates the footer from the rest of the page.

I’ve covered this in detail before, but here’s the quick version. You’ll need to grab a snippet of code from a resource like Websense Pro. Once you have it:

Go to Online Store > Themes > Edit Code.
Under Sections, click Add a new section. Name it something like divider-section.liquid.
Paste the code you copied and hit Save.
Now go back to the customizer. You’ll see a new section called Advanced Shape Divider that you can add above the footer. Choose your shape—waves, clouds, jagged edges—whatever fits your vibe.

Match the background color to your footer, and the shape color to the section above. It blends in perfectly and adds that extra layer of polish.

Step 9: Tweak the Fonts
Finally, click the Theme Settings gear icon and go to Typography. Swap out the default heading font for something with personality. I personally love Abril Fatface—it’s bold and adds character without trying too hard.

Click Select and then Save.
