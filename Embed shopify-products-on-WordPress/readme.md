In this step-by-step guide, I’ll walk you through exactly how to install, set up, and use this new plugin.

# What You’ll Need Before Starting:
A live WordPress website with admin access.
An active Shopify store (a development store works perfectly).
The **“Sell on WordPress”** plugin for WordPress.
The **“Sell on WordPress”** sales channel for Shopify.

# Step 1: Install the Plugin on Your WordPress Site
First, we need to get the plugin onto your WordPress dashboard.

Log in to your **WordPress Admin** panel.
Navigate to **Plugins > Add New Plugin.**
Click on Upload Plugin.
Choose the plugin file you downloaded and click **Install Now.**
After installation, click **Activate Plugin.**
You should now see a new Shopify section in your dashboard, confirming the plugin is active.

# Step 2: Add the “Sell on WordPress” Channel in Shopify
Now, we need to connect your Shopify store to the plugin.

Log in to your **Shopify Admin** dashboard.
Go to the **Shopify App** Store and search for **“Sell on WordPress”.**
Look for the official app **created by Shopify** (check the logo to be sure) and click on it.
Click **“Add app”** and then **“Install”** to add it as a sales channel to your store.

# Step 3: Connect Shopify to WordPress
This is the crucial step where we link the two platforms together.

After installing the app in Shopify, you will be presented with a unique **Shopify Access Token. Copy this token.**
Go back to your **WordPress Admin** panel and find the Shopify settings (usually under a new “Shopify” menu item).
Look for the field labeled **“Shopify Access Token”.**
**Paste the token** you copied from Shopify and click **“Connect”.**
A success message should appear, confirming that your WordPress site is now connected to your Shopify store.

# Step 4: Embed Products into Your WordPress Pages
With the connection live, you can now start adding products directly into your pages using the WordPress block editor (Gutenberg).

**How to Embed a Single Product:** 
Create a new **Page** or edit an existing one in WordPress.
In the Gutenberg editor, click the **“+”** button to add a new block.
Search for **“Shopify”.** You will see two new blocks: **“Shopify Product”** and **“Shopify Collection”.**
Select **“Shopify Product”.**
A dropdown menu will appear, showing all the products from your connected Shopify store. Select the product you want to display.
Click **“Publish”** or **“Update”** your page.

Now, visitors to that page will see the product with its image, title, and a **“Quick Shop”** button. Clicking “Quick Shop” opens a stylish pop-up where users can select variants and click **“Buy Now”** to be taken directly to Shopify’s secure checkout.

# How to Embed a Product Collection:
Inside the page editor, add a new block and search for **“Shopify Collection”.**
Select the block and choose which collection you want to display from the dropdown (e.g., “All”).
You can customize the display by changing the **“Max products per row”** and **“Max products per page”.**
Publish your changes.
The collection will display on your page with pagination, and each product will have the same “Quick Shop” functionality.

# Final Result & Key Features
Once set up, you and your visitors will enjoy:

**Seamless Integration:** Products and collections look like a native part of your WordPress site.
**Quick Shop Pop-ups:** Customers can view details and select options without leaving the page.
**Secure Shopify Checkout:** All purchases are processed through Shopify’s trusted and optimized checkout system.
**Custom Styling:** You can add custom CSS classes to the blocks to match your site’s design perfectly.

**A Quick Note for Elementor Users**
This tutorial uses the standard WordPress Gutenberg editor. **This plugin does not currently work with Elementor widgets.** If you build your pages primarily with Elementor, you’ll need to wait for a dedicated solution or use the plugin in a Gutenberg block within an Elementor page.

# Conclusion
The new “Sell on WordPress” plugin is a powerful and simple solution for merging the content management strength of WordPress with the e-commerce excellence of Shopify. It eliminates the need for a costly and complex migration, letting you enhance your existing site with a full-featured store in just a few steps.
